# Installing

## Extension
* Go to extension page `vivaldi://extensions` (shortcut ctrl+shift+e)
* Enable Developer mode (top right corner)
* Load unpacked -> select `dist/chrome`

## Hook
`sudo ./installBrowserHook.sh` and select target Vivaldi installation.
